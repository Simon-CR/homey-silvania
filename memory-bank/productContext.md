# Product Context

## Overview

This project is a Homey app to support Sylvania (LEDVANCE) Smart+ Zigbee devices. The initial focus is on the Sylvania Smart+ A19 RGBW bulb (Model 75564).

## Core Features

- **Device Support**: Sylvania Smart+ A19 RGBW (75564)
- **Capabilities**:
    - On/Off
    - Dimming
    - RGB Color Control
    - Color Temperature Control (142-555 mireds)
- **Zigbee Integration**: Uses `homey-zigbeedriver` for standard Zigbee cluster communication.

## Technical Stack

- **Platform**: Homey (Athom)
- **Language**: Node.js
- **Framework**: Homey SDK v3, homey-zigbeedriver
- **Protocol**: Zigbee
