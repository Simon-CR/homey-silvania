Sylvania Zigbee App for Homey

This app adds support for Sylvania Smart+ Zigbee devices to Homey.

Supported Devices:

1. Sylvania Smart+ A19 RGBW (Model 75564)
   - On/Off, Dimming, Color (RGB), Color Temperature

2. Sylvania Smart+ Flex Strip RGBW
   - On/Off, Dimming, Color (RGB), Color Temperature

3. Sylvania Smart+ Smart Plug
   - On/Off

4. Sylvania Smart+ BR30 RGBW
   - On/Off, Dimming, Color (RGB), Color Temperature

If your device is not recognized, please report it on GitHub.
